## 2024-05-18 - Improve Newsletter form & Social links accessibility in Footer
**Learning:** Adding labels to visual form elements using the `sr-only` class is essential for screen reader users and improves general accessibility without altering the visual design. Similarly, adding `aria-label` to anchor tags containing only SVG icons is important for screen reader context. Incorporating `focus:outline-none focus:ring-2 focus:ring-blue-500` ensures a visible focus state during keyboard navigation across browsers.
**Action:** Consistently ensure inputs have explicit `<label>` elements or robust `aria-label` descriptions. Additionally, wrap standalone icons acting as interactive elements with clear `aria-label` text, and verify keyboard navigation focus styles are explicitly defined.
